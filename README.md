
# Excel Stok Takip Sistemi

Bu proje, endüstri mühendisliği bakış açısıyla hazırlanmış
bir Excel tabanlı stok takip ve karar destek çalışmasıdır.

## Kullanılan Özellikler
- IF ve iç içe IF formülleri
- Koşullu biçimlendirme (stok seviyelerine göre renkler)
- Stok durumuna göre otomatik karar üretimi
- Toplam stok değeri hesaplama
- Yönetici özet alanı

## Amaç
Stok seviyelerini hızlıca analiz etmek,
kritik ürünleri tespit etmek ve karar sürecini kolaylaştırmak.

## Kullanılan Araç
- Microsoft Excel

## Not
Bu çalışma eğitim ve portföy amaçlı hazırlanmıştır.
